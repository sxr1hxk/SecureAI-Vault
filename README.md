# 🔐 SecureAI Vault – Intelligent Password Manager

## 📌 Overview

This project is a secure password manager built using Flask and Python. It focuses on encryption, authentication, and safe credential storage.

## 🚀 Features

* Secure password storage 🔑
* User authentication 🔐
* Encryption-based security
* Simple and user-friendly interface

## 🛠️ Tech Stack

* Python
* Flask
* HTML/CSS
* SQLite

## ⚙️ How to Run

1. Download the project
2. Install dependencies
3. Run:
   python app.py
4. Open browser:
   http://127.0.0.1:5000/

## 🔒 Security Features

* Password encryption
* Input validation
* Secure storage
